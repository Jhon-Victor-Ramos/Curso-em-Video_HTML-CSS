Testando o uso de um iframe

O tamanho padrão de um iframe é: 300x150 px. Podemos mudar a altura dele, fazendo o uso da tag height e, também, a largura dele colocando a tag width dentro da tag do próprio iframe.

Podemos mudar a altura e a largura de um iframe, fazendo uso do CSS. Caso eu tenha um iframe estilizado da seguinte forma:
< iframe iframe src="" frameborder="0" height="900px" width="1320px" >< /iframe > E definir um CSS para um iframe, a configuração definida no CSS irá prevalecer. O CSS é soberano.

Existem outro parâmetro para se usar no iframe também, o scrolling que está direcionado à barra de rolagem. Funciona da seguinte forma:

scrolling="auto" ↔ Caso o conteúdo dentro do iframe exceda o tamanho do mesmo, a barra de rolagem será ativada.
scrolling="yes" ↔ Mesmo que o conteúdo caiba totalmente dentro do iframe, a barra de rolagem estará habilitada.
scrolling="no" ↔ A barra de rolagem estará desabilitada totalmente.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Já o frameborder - tag usada dentro da tag do iframe - possui apenas dois valores, o 0 (zero) e o 1 (um). Sendo o 0 para definir que o iframe não terá bordas, e o 1 para que ele tenha borda, apenas de um pixel. Caso você queira definir uma borda diferente, basta usar o border no CSS.
